Urban Grind: Gym Management System

Complete Technical Documentation & System Overview

1. System Overview

The Urban Grind platform is an enterprise-grade digital solution designed to streamline gym operations. It provides a robust management layer for admins, a specialized toolset for trainers, and a personalized experience for members.

2. Technology Stack

Frontend Architecture

  • React.js (Vite): For high-speed UI rendering.
  • Context API: Managing global state (Auth, Members, Plans).
  • Vanilla CSS: Customizable variable-based styling.

Backend Infrastructure

  • Django & DRF: RESTful API for secure data handling.
  • JWT Authentication: Token-based stateless security.
  • Django Signals: Automated background workflows.

3. Core Features

  • Multi-Role Access: Personalized views for Admin, Trainer, and Member.
  • Subscription Tiers: Logic for Basic, Gold, and Diamond memberships.
  • Broadcast System: Mass-email marketing with plan-based filtering.
  • Theme Engine: Gradient and Premium themes with persistence.
  • Attendance Tracker: Analytical views for member activity.
  • Communication Suite: Integrated email and in-app alerts.

4. Key Workflows

Automated Member Onboarding

When a member registers, the system executes a transactional save involving the User, Profile, and Member models. Upon completion, a Welcome Email is automatically dispatched via the Django backend utility.

The Notification-Email Sync

To ensure members never miss an update, we implemented a Post-Save Signal. Whenever an in-app notification is created (for plan reminders, trainer assignments, etc.), the system automatically duplicates the message and sends it as an official email.

Segmented Marketing Broadcasts

Admins can choose target audiences (e.g., "Diamond Members Only"). The system dynamically filters active emails and sends a batch broadcast using a customized HTML template, ensuring 100% targeting accuracy.

5. Data Architecture

Model Purpose
User Core authentication and credentials.
Profile Extended data (Role: ADMIN, TRAINER, MEMBER).
Member Subscription details, expiry, and trainer links.
Broadcast Marketing logs and history.

6. Setup & Deployment

Backend

1. Install requirements: pip install -r requirements.txt
2. Run migrations: python manage.py migrate
3. Start Server: python manage.py runserver

Frontend

1. Install dependencies: npm install
2. Run Dev: npm run dev
